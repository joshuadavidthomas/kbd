# kbd-derive

Derive macros for [`kbd`](https://crates.io/crates/kbd) — `#[derive(Bindings)]` for declarative hotkey registration, `#[hotkey(...)]` attributes, `#[flatten]` for composition, and compile-time hotkey string validation.
